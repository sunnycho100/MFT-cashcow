# Research Index

This folder is the starting record for `v3`.

Files in this folder:
- `01-strategy-map.md`: what strategies are worth building, how they fit `v3`, and paper-reported benchmark numbers.
- `02-api-catalog.md`: Kraken and supporting data APIs we can use for live trading and signal generation.
- `03-v3-architecture.md`: recommended `v3` service layout, data transmission layer, and coding sequence.
- `08-quant-fundamentals.md`: higher-confidence research on what actually matters in crypto quant and what data should come next.
- `09-practitioner-scan.md`: lower-confidence practitioner / X-style scan used for hypothesis generation only.
- `10-return-max-v3-roadmap.md`: research-backed roadmap for a higher-return `v3` branch.

Scope notes:
- Benchmarks below are research or paper-reported results, not validated repo results.
- Any paper with very high Sharpe / annualized returns should be treated as a hypothesis source, not a deployment target.
- Before live use, every claimed edge still needs our own 3-year backtest and paper-forward validation.
